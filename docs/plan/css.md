
## 什么事是BFC
- BFC的字面意思为块级格式上下文，是一个独立的渲染区域，BFC的内部的子元素不会受到外界元素的影响
- 触发BFC的方式
1. HTML根元素
2. 元素浮动
3. 元素绝对定位、固定定位
4. overflow不为visible
5. inline-lock、flex等
